## Base image tags

The tags of `public.ecr.aws/lambda/java` (`:25`, `:21`, …) are mutable pointers: each one always refers to the **latest build** of its Java version. When AWS publishes a new build, the same tag moves to the new content – the tag stays, the digest behind it changes. A tag alone therefore does not identify a specific image; only its current digest does (pin it as `<image>:<tag>@sha256:<digest>` for a reproducible build).
